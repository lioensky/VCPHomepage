# 更新日志总览

本文依据 [`VCP聚合文档.md`](VCP聚合文档.md) 中已明确记录的开发时间线整理，按**最新在前**的倒序方式展示，方便先看最近更新，再回溯历史演进。

---

## 最新更新

### 2026-04-05 · 官网文档中心首版上线

官网现已具备文档分页、Markdown 自动导入、Frontmatter 解析、Mermaid / LaTeX 渲染、更新时间排序与更新日志专区等基础能力，对外展示体系开始成型。
同日，VChat 的 preload 管理完成重构，改为基于 role + key 的动态鉴权与载入工厂模式，便于进一步优化权限隔离、模块装载与内存管理。

### 2026-04-04 · 浪潮 RAG 7.5 与官网上线

浪潮升级到 7.5，围绕动量机制、逻辑张力探测与引力弹弓效应继续增强跨域联想能力；同日 [`https://www.vcptoolbox.com/`](https://www.vcptoolbox.com/) 官网正式上线。

### 2026-04-03 · 上下文附件管理器重构

上下文附件获得独立 fileapi 可视化索引，支持对图片、音频、视频、文档、代码进行拖拽、编辑、合并与排序，附件管理能力显著提升。

### 2026-04-02 · PreText.js 接入

文本渲染类 DOM 引入基于微型神经网络的布局拟合方案，明显降低文本布局计算开销，长文本渲染效率大幅提升。

## 2026-03

### 2026-03-31 · 群聊与前后端耦合继续增强

VCPGroupChat 引擎重构为高级策略模式；后端语义向量引力场开始通过 WS-Bridge 暴露给更多分布式能力；前端 ChatToVCP 也可传递更多可靠字段。

### 2026-03-30 · API 管理与桌面能力继续打通

打通 VCP-NewApi 与 VCPToolbox 之间的 API 通讯交换，Agent 开始具备更强的后台 API 管理能力；同时进一步完善桌面端调用底层系统能力的通路。

### 2026-03-29 · 梦系统底层引擎重做

梦系统从早期可行性验证进入底层引擎重构阶段，开始模拟更接近人类梦境的短期、中期、长期记忆交叉放电联想机制。

### 2026-03-28 · 前端设计语言升级

VChat 全局子应用设计语言重构，引入液态玻璃风格与更轻量的磨砂 Blur 管线，统一整体视觉表现。

### 2026-03-25 · 稳定性与性能维护

优化浪潮算法在异步、并发与数据库安全方面的可靠性；同时修复桌面相关已知问题，继续优化渲染器在桌面场景下的 RAM 与 CPU 占用。

### 2026-03-23 · VCPDesktop 深度扩展

进一步优化桌面 HTML 占用性能，增强 Widget 沙箱与权限体系，统一 [`VCPAppAPI`](VCP聚合文档.md)，让 AI 通过更少配置即可调用桌面程序、VChat 子应用与分布式能力。

### 2026-03-22 · VCPDesktop 大更新

新增壁纸编辑、图标编辑、桌面锁定、多显示器兼容、桌面预设管理器、Docker 栏、折叠应用文件夹、桌面遥控器插件等能力，并逐步完善桌面生态。

### 2026-03-21 · VCPDesktop 落地

VCPDesktop 作为新的桌面级子系统正式上线，允许 AI 使用流式语法直接向桌面推送 Widget、动态 HTML 与交互式内容，打通 AI 与桌面 UI 的边界。

### 2026-03-20 · VSearch+ 上线

VSearch 升级出增强版本，聚合多个搜索能力并统一总结返回，提升搜索结果覆盖面与质量。

### 2026-03-19 · Memo 管理中心升级

基于浪潮 V7，前端支持记忆神经节点云图可视化、关联记忆高亮、多文档批量整理与可视化工作台，记忆管理体验显著增强。

### 2026-03-18 · 全局向量缓存与音乐系统重构

统一后端向量缓存内存管理器；新增全局工具调用语法；VChat 音乐播放器重构为更专业的后端播放器遥控架构，并持续升级 EQ、重采样与 WASAPI 管线。

### 2026-03-17 · 大扫除与细节修复

修复大量迷你 Bug，重新整理分布式系统 CSS，并对界面间距和统一性进行像素级微调。

### 2026-03-16 · 占位符变量管理器统一

后端统一占位符变量管理器，支持内存预览、即时编译、动态修改、动态向量微调与阈值控制；同时推进划词小助手和日记管理器的 Rust 化重构。

### 2026-03-14 · 手动神经元标记语法上线

新增 `[@tag]` 与 `[@!tag]` 语法，让 AI 可以在输出中主动参与构建 TagMemo 神经网络；同时新增 `::Truncate0.25` 等更细粒度的日记本控制语法。

### 2026-03-13 · 通知系统管线重构

VCPLOG 通知系统重构后，用户可在任意前端通知栏中直接回复 AI 的工具调用结果，也能进行允许 / 拒绝等审批动作。

### 2026-03-12 · 浪潮 V7 更新

引入有向递归 Tag 矩阵神经元共现技术，通过预计算残差与序位势能，进一步增强神经元放电路径控制与长程联想能力。

### 2026-03-11 · 上下文语块智能感知切分器上线

VCP 后端开始自动感知上下文数组中的信息来源，按规则把通知栏、工具返回、连续数组等内容切入更合适的角色位，增强 Agent 对上下文结构的掌控力。

### 2026-03-11 · 清晨 6:11 · 浪潮 V6 升级

浪潮升级到 V6，开始引入对 LIF 神经元放电模型的拟合能力，同时新增 `::TimeDecay`、`::Rerank+0.7`、`::RoleValve@User>x` 等高级日记本语法，并将 LightMemo 与 MeshMemo 合并。

### 2026-03-10 · 异步协议升级到 V2

异步工具调用的进度与结果除了占位符追踪，还会经由通知栏主动推送；[`VCPAgentAssistant`](VCP聚合文档.md) 也获得更丰富的多类型插件基建能力。

### 2026-03-09 · 多模态日记与分布式预处理推进

VCP 日记开始支持附加任意形式文件并自动构建多模态索引；预消息处理器支持分布式部署，进一步强化摄像头、桌面窗口等多端感知能力。

### 2026-03-06 · 划词小助手底层完全 Rust 重构

实现全平台兼容的窗口滑动监控、文件选取监控与文字选取监控，进一步增强 Agent 与用户对系统内容的感知交互能力。

### 2026-03-02 · VCPAuthNet 上线

支持不同 VCP 用户之间通过安全授权验证码构建公网互联，实现跨用户 Agent 通讯与论坛互联。

## 2026-02

### 2026-02-27 · 超栈追踪 2.0 上线

分布式文件、节点、插件、插件管理器、Agent 与 Agent 间通讯开始对上层完全透明；同时 Vexus 和浪潮算法进一步升级，全面支持多模态文件。

### 2026-02-25 · 全局下载、网页访问与分布式协议统一

统一下载功能、网页访问重定向、分布式文件管理、分布式 CLI、窗口感知与 SOM 操作模块，减少系统内部分裂实现。

### 2026-02-24 · 工具回调同构完成

40 多个官方同步 / 异步插件统一为标准 OpenAI API 数组结构，错误统一进 `text` 字段，数据统一进 `base64` 字段，并全面 MD 化回调内容。

### 2026-02-22 · 电脑控制算法与模型管理增强

优化串行指令、文字输入、滚动、键盘控制与多窗口并发控制；17 个静态插件适配上下文折叠协议 V2；同时新增高级动态模型路由与模型管理面板。

### 2026-02-21 · VCP-SOM 落地

VCP 自研纯数学窗口操作算法正式成型，Agent 可以几乎不依赖上下文图像，仅靠语义描述感知和控制电脑窗口。

### 2026-02-20 · 标准上下文折叠协议 V2 实现

支持动态折叠话题偏离内容、静态插件返回结果与更多上下文信息，实现更细粒度的 Token 控制与上下文聚焦。

### 2026-02-18 · 梦系统沙箱基建完成

Agent 在梦中可以调用工具、看到真实结果，但这些结果默认不会直接落到现实世界，用户可在梦面板中决定是否同步。

### 2026-02-16 · 中央服务继续统一

统一本地文件 / 字段搜索类事务到中央服务，构建搜索队列与索引暂存；优化记忆召回策略；同时将前端配置写入全面委托到中央序列化流程。

### 2026-02-15 · 梦系统开始测试

休眠期 Agent 具备做梦概率，梦记忆被写入独立梦记忆区，默认不会被普通被动回忆系统直接召回。

### 2026-02-12 · 语义级日记分类与聚合检索增强

新增日记语义级智能文件夹分类与自动化索引重建；同时支持聚合检索语法，增强多日记本联合召回能力。

### 2026-02-06 · Vmusic 底层彻底重构

采样与解码改为更激进的统一汇编实现，通信层迁移到 Rust，并补齐更完整专业的音乐播放器 UI 与管理能力。

### 2026-02-02 · TagMemo V5 公布

偏振语义舵进入公开版本，并加入霰弹检索与相控阵雷达技术，大幅提升逻辑捕获与降噪能力。

## 2026-01

### 2026-01-29 · BilibiliFetch / YoutubeFetch 重构

Agent 获得更完整的视频平台能力，包括检索、关注、字幕 / 视频 / 音频下载、多时间点截图、评论与上传等。

### 2026-01-27 · ToolPayload 解析器优化

全局 ToolPayload 解析器优化后，插件返回内容更适合阅读，同时引入部分 URL 重定向解析器处理特殊返回。

### 2026-01-23 · 文件管理与划词小助手增强

显著提升文件批处理、队列处理、回退与报错审查能力；划词小助手也支持从任意文件夹右键呼出，扩大生产力边界。

### 2026-01-18 · TagMemo V3 持续优化

继续强化 CoreTag 聚类、动态能量裁剪、共现关联、上下文 K 震荡与 Rust 端精度 / 防抖策略。

### 2026-01-17 · TagMemo V3 初步成型

引入上下文意图差值波动、上下文熵增追踪、EPA 投影、残差金字塔、远端逻辑寻路等一系列仿生记忆增强算法。

### 2026-01-14 · F64 全链路无损采样引入

基于 C.libsox 和汇编魔改，引入 F64 Polyphase 采样、SIMD 环形加速、Buffer 窗口控制与相位时钟控制。

### 2026-01-12 · 音频解码器大幅优化

音频系统继续引入 FIR-EQ、多级采样仿真、动态噪音整形与更好的时钟同步与底层 buffer 管理。

### 2026-01-11 · 中央消息处理器重构

消息处理器全面解耦，建立独立 socket 消息追踪、缓存与工具调用 Promise 容器，显著提升流式稳定性。

### 2026-01-09 · 元思维链升级到 V2

引入二维矩阵 step 递归的隐性语法传递，更深度引导模型推理智慧，新增 27 个插件。

### 2026-01-05 · DOM 墓碑暂存机制上线

流式渲染器支持冻结视野范围外 DOM 而非销毁，避免高层对话中的状态丢失与重复渲染问题。

### 2026-01-03 · IPC 动态内存加载技术上线

VChat 全局 IPC 引入动态加载和自动卸载机制，显著提升启动速度、性能和多子应用协同体验。

### 2026-01-01 · 后端管理面板重构为 React 响应式

后端面板开始支持更现代的交互、语义级检索、多分辨率适配与手机布局。

### 2025-12-19 · 上下文折叠技术扩展

### 2025-11-18 · TagMemo 与 Vexus 上架

VCP 的长期记忆系统进入关键阶段，TagMemo 与 Vexus 形成记忆与向量数据库底座。

### 2025-10-12 · Agent 自主意识与论坛上线

Agent 开始具备更强的自主行动能力，心流锁与 VCP 论坛一起上线。

### 2025-09-25 · 超级流式差分渲染器完成 Rust 统一重构

流式渲染器成为可插拔核心底层依赖，可稳定以高帧率渲染复杂内容。

### 2025-09-15 · RAG 日记本语法落地

四大日记本与多种语法成型，VCP 记忆系统开始真正进入体系化阶段。

### 2025-08-25 · 子应用框架全面铺开

Vnote、Vmusic、Vcanvas、Vcli 等子应用框架开始形成完整生态。

### 2025-08-07 · 分布式服务器、超栈追踪协议、VCPFileAPI 落地

VCP 开始具备跨节点工具调用与透明文件访问能力，并新增 61 个插件。

### 2025-06-17 · VChat 落地

官方前端客户端成型，成为后续复杂渲染、多媒体与桌面能力的承载核心。

### 2025-05-13 · 六大插件协议与服务器管理器落地

VCP 服务器与插件体系正式成型，建立 [`static`](VCP聚合文档.md)、[`messagePreprocessor`](VCP聚合文档.md)、[`synchronous`](VCP聚合文档.md)、[`asynchronous`](VCP聚合文档.md)、[`service`](VCP聚合文档.md)、[`hybridservice`](VCP聚合文档.md) 六大协议，新增 27 个插件。

### 2025-04-12 · VCP 计划正式启动

VCP 从构思阶段进入正式开发阶段。

### 2025-02-05 · VCP 计划开始构思

项目正式进入概念设计期，并开始围绕 AI 中间层、记忆、插件和前后端协同展开规划。
